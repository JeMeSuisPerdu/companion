<h1> Compagnon (Front-End)</h1>

<p><strong>Compagnon</strong> est une application <strong>React</strong> servant d’interface front-end pour la plateforme d’accompagnement des étudiants en situation de handicap.<br>
Elle communique avec l’API <strong>companion_api</strong> pour gérer l’inscription et la connexion des utilisateurs.</p>

<hr>

<h2> Fonctionnalités principales</h2>
<ul>
  <li>Interface utilisateur développée avec <strong>React</strong></li>
  <li>Connexion directe avec <strong>companion_api</strong></li>
  <li>Gestion de l’inscription et de la connexion des utilisateurs</li>
  <li>Architecture claire et modulaire, prête pour l’évolution du projet</li>
</ul>

<hr>

<h2> Stack technique</h2>
<ul>
  <li><strong>Framework :</strong> React 18+</li>
  <li><strong>Backend :</strong> companion_api (API REST JAVA SPRING BOOT)</li>
</ul>
